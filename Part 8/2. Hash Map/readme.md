### Hash Map
- Learning Objectives
1. You're familiar with the concept of a hash map and know how one works.
2. You know how to use Java's hash map: you know how to create one, add information to it and retrieve information from it.
3. You can describe situations where using a hash map could be useful.
4. You know how to use a hash map as an instance variable.
5. You know how to go through keys and values of a hash map using the for-each loop.
